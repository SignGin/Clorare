# **Clorare**
- 프로젝트명 : Clorare (Clothes Random Recommendation Project) <br>
- 기획 및 제작 : 이유찬(Front-end), 주기현(Back-end)<br>
- 주제 : 옷 랜덤 추천
  - 현재 날씨에 따라 등록된 데이터에서 옷을 추천해주는 서비스입니다. <br>
- 사용 툴 : Python, Django, MySQL, JavaScript, Next.js
